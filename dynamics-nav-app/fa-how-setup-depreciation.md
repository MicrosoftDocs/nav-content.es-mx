## <a name="to-create-a-depreciation-book"></a><span data-ttu-id="69337-110">Para crear un libro de amortización</span><span class="sxs-lookup"><span data-stu-id="69337-110">To create a depreciation book</span></span>  
<span data-ttu-id="69337-111">En un libro de amortización de activos fijos puede especificar la forma en que se amortizarán los activos fijos.</span><span class="sxs-lookup"><span data-stu-id="69337-111">In a fixed asset depreciation book, you specify how fixed assets are depreciated.</span></span> <span data-ttu-id="69337-112">Para aplicar distintos métodos de amortización, puede configurar varios libros de amortización.</span><span class="sxs-lookup"><span data-stu-id="69337-112">To accommodate various methods of depreciation, you can set up multiple depreciation books.</span></span>  
1.  <span data-ttu-id="69337-113">En la esquina superior derecha, seleccione el icono **Buscar página o informe**, escriba **Libros amortización** y, a continuación, seleccione el vínculo relacionado.</span><span class="sxs-lookup"><span data-stu-id="69337-113">In the top right corner, choose the **Search for Page or Report** icon, enter **Depreciation Books**, and then choose the related link.</span></span>
2. <span data-ttu-id="69337-114">En la ventana **Lista libros amortización**, elija la acción **Nuevo**.</span><span class="sxs-lookup"><span data-stu-id="69337-114">In the **Depreciation Books List** window, choose the **New** action.</span></span>
3. <span data-ttu-id="69337-115">En la ventana **Ficha libros amortización**, rellene los campos según sea necesario.</span><span class="sxs-lookup"><span data-stu-id="69337-115">In the **Depreciation Book Card** window, fill in the fields as necessary.</span></span> <span data-ttu-id="69337-116">Seleccione un campo para obtener una breve descripción del campo o el enlace a información adicional.</span><span class="sxs-lookup"><span data-stu-id="69337-116">Choose a field to read a short description of the field or link to more information.</span></span>

    <span data-ttu-id="69337-117">**Nota**: Puede registrar transacciones de activos en las ventanas **A/F Diario general** o **Diario activos fijos**, dependiendo de si las transacciones son para la información financiera o la gestión interna.</span><span class="sxs-lookup"><span data-stu-id="69337-117">**Note**: You can record fixed asset transactions in the **Fixed Asset G/L Journal** window or in the **Fixed Asset Journal** window, depending on whether the transactions are for financial reporting or for internal management.</span></span> <span data-ttu-id="69337-118">Vaya al siguiente paso para indicar qué tipo de diario se utiliza para las distintas actividades de activos fijos de forma predeterminada.</span><span class="sxs-lookup"><span data-stu-id="69337-118">Follow the next step to define which type of journal is used for the different fixed asset activities by default.</span></span>
4. <span data-ttu-id="69337-119">En la ficha desplegable **Integración**, seleccione la casilla de las transacciones de cada actividad de activo que desee registrar en la ventana **A/F Diario general**.</span><span class="sxs-lookup"><span data-stu-id="69337-119">On the **Integration** FastTab, select the check box for each fixed asset activity whose transactions you want to post using the **Fixed Asset G/L Journal** window.</span></span>

## <a name="to-assign-a-depreciation-book-to-a-fixed-asset"></a><span data-ttu-id="69337-121">Para asignar un libro de amortización a un activo fijo</span><span class="sxs-lookup"><span data-stu-id="69337-121">To assign a depreciation book to a fixed asset</span></span>  
1. <span data-ttu-id="69337-122">En la esquina superior derecha, seleccione el icono **Buscar página o informe**, escriba **Activos** y, a continuación, seleccione el vínculo relacionado.</span><span class="sxs-lookup"><span data-stu-id="69337-122">In the top right corner, choose the **Search for Page or Report** icon, enter **Fixed Assets**, and then choose the related link.</span></span>

3. <span data-ttu-id="69337-124">En la ficha desplegable **Ficha libros amortización**, rellene los campos según sea necesario.</span><span class="sxs-lookup"><span data-stu-id="69337-124">On the **Depreciation Book** FastTab, fill in the fields as necessary.</span></span>

    <span data-ttu-id="69337-138">Si rellena este campo, los campos de amortización de los nuevos libros de amortización de activos fijos contendrán la misma información que los campos correspondientes del libro desde el que va a copiarlos.</span><span class="sxs-lookup"><span data-stu-id="69337-138">If you fill in this field, the depreciation fields in the new fixed asset depreciation books will contain the same information as the corresponding fields in the fixed asset depreciation book that you copy from.</span></span> <span data-ttu-id="69337-139">Deje este campo en blanco si desea crear libros de depreciación de activos fijos con los campos de depreciación vacíos.</span><span class="sxs-lookup"><span data-stu-id="69337-139">Leave the field blank if you want to create new fixed asset depreciation books with empty depreciation fields.</span></span>  
6. <span data-ttu-id="69337-140">En la ficha desplegable **Activo**, puede establecer un filtro para seleccionar los activos para los que desee crear los libros de amortización de activos.</span><span class="sxs-lookup"><span data-stu-id="69337-140">On the **Fixed Asset** FastTab, you can set a filter to select the assets that you want to create the fixed asset depreciation books for.</span></span>
