3. <span data-ttu-id="39597-110">En la ventana **Atributos de producto**, rellene los campos según sea necesario.</span><span class="sxs-lookup"><span data-stu-id="39597-110">In the **Item Attribute** window, fill in the fields as necessary.</span></span> <span data-ttu-id="39597-111">Seleccione un campo para obtener una breve descripción del campo o el enlace a información adicional.</span><span class="sxs-lookup"><span data-stu-id="39597-111">Choose a field to read a short description of the field or link to more information.</span></span>

<span data-ttu-id="39597-112">**Nota**: Si selecciona **Opción** en el campo **Tipo**, puede seleccionar la acción **Valores de atributo de producto** para crear valores de atributo de producto.</span><span class="sxs-lookup"><span data-stu-id="39597-112">**Note**: If you select **Option** in the **Type** field, then you can choose the **Item Attribute Values** action to create values for the item attribute.</span></span> <span data-ttu-id="39597-113">Para obtener más información, consulte la sección "Para crear valores de atributo de producto del tipo Opción".</span><span class="sxs-lookup"><span data-stu-id="39597-113">For more information, see the "To create values for item attributes of type Option" section.</span></span>

## <a name="to-assign-item-attributes-to-items"></a><span data-ttu-id="39597-119">Asignar un atributo de producto a productos</span><span class="sxs-lookup"><span data-stu-id="39597-119">To assign item attributes to items</span></span>
1. <span data-ttu-id="39597-120">En la esquina superior derecha, seleccione el icono **Buscar página o informe**, escriba **Productos** y, a continuación, seleccione el enlace relacionado.</span><span class="sxs-lookup"><span data-stu-id="39597-120">In the top right corner, choose the **Search for Page or Report** icon, enter **Items**, and then choose the related link.</span></span>
2. <span data-ttu-id="39597-121">En la ventana **Producto**, seleccione el producto al que quiere asignarle un atributo de producto y, a continuación, seleccione la acción **Atributos**.</span><span class="sxs-lookup"><span data-stu-id="39597-121">In the **Items** window, select the item that you want to assign item attributes to, and then choose the **Attributes** action.</span></span>
3. <span data-ttu-id="39597-122">En la ventana **Valores de atributo de producto**, seleccione la acción **Nuevo**.</span><span class="sxs-lookup"><span data-stu-id="39597-122">In the **Item Attribute Values** window, choose the **New** action.</span></span>

6. <span data-ttu-id="39597-126">Para los atributos de producto del tipo Opción, seleccione el botón AssistEdit en el campo **Valor** y seleccione un valor de atributo de producto.</span><span class="sxs-lookup"><span data-stu-id="39597-126">For item attributes of type Option, choose the AssistEdit button in the **Value** field and select an item attribute value.</span></span> <span data-ttu-id="39597-127">De forma alternativa, elija la acción **Nuevo** para crear primero un nuevo valor de atributo de producto como se explica en la sección "Para crear valores de atributo de producto del tipo Opción".</span><span class="sxs-lookup"><span data-stu-id="39597-127">Alternatively, choose the **New** action to first create a new item attribute value as explained in the "To create values for item attributes of type Option" section.</span></span>

## <a name="to-filter-by-item-attributes"></a><span data-ttu-id="39597-140">Para filtrar por atributos de producto</span><span class="sxs-lookup"><span data-stu-id="39597-140">To filter by item attributes</span></span>
1. <span data-ttu-id="39597-141">En la esquina superior derecha, seleccione el icono **Buscar página o informe**, escriba **Productos** y, a continuación, seleccione el enlace relacionado.</span><span class="sxs-lookup"><span data-stu-id="39597-141">In the top right corner, choose the **Search for Page or Report** icon, enter **Items**, and then choose the related link.</span></span>

6. <span data-ttu-id="39597-149">En la ventana **Especificar valor de filtro**, en el campo **Condición**, haga clic en la flecha desplegable y seleccione una condición.</span><span class="sxs-lookup"><span data-stu-id="39597-149">In the **Specify Filter Value** window, in the **Condition** field, choose the drop-down arrow and select a condition.</span></span>
7. <span data-ttu-id="39597-150">En el campo **Valor**, introduzca un valor de atributo para filtrar los productos.</span><span class="sxs-lookup"><span data-stu-id="39597-150">In the **Value** field, enter an attribute value to filter items by.</span></span>

    <span data-ttu-id="39597-151">**Ejemplo**: Para filtrar los productos en que la descripción del material empieza por "azul", rellene los campos como se indica a continuación: campo **Atributo**: descripción de material; campo **Condición**: empieza por; campo **Valor**: azul.</span><span class="sxs-lookup"><span data-stu-id="39597-151">**Example**: To filter on items where the material description begins with "blue", fill in the fields as follows: **Attribute** field: Material Description, **Condition** field: Begins With, **Value** field: blue.</span></span>
